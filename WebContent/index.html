Você esta utilizando um Servidor de Web Services em REST!!!


URI de Web Services implementados:

  • rest/hello/{nome}
  • rest/frete/getValor/{estado}
  • rest/frete/clientes
  • rest/cliente -- Implementação de todos os métodos:
    • GET: rest/cliente, rest/cliente/{id}, rest/cliente/gerar
    • POST: rest/cliente
    • PUT: rest/cliente/{id}
    • DELETE: rest/cliente/{id}
